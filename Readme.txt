Original code by David Hopkins (https://hackaday.io/Hopo28)

This package contains everything required to make the 7 segment neopixel clock, except the stock Adruino IDE. 

This is known to work with Arduino version 1.6.4, as of 28 September 2015.

Also included is a Frtizing diagram for the wiring and the Sketchup file for the clock face.
